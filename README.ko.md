# 한눈에 보는 성경 이야기 (Bible in One Scroll)

창조부터 회복까지, **성경 전체를 한 편의 이야기로** 스크롤하며 한눈에 이해하도록 만든 모바일 친화적 단일 페이지 사이트입니다.

🔗 **https://one-scroll-bible.com/**

## 무엇을 만든 것인가
이것은 **성경 번역본이 아니고**(번역은 성서공회·위클리프가 합니다), **성경 앱도 아닙니다**(유버전이 합니다). 그 앞단의 **‘입구(on-ramp)’** 입니다.

> **세계 대부분의 사람이 자기 모국어로, 5분 안에, 성경 66권의 큰 그림(창조→타락→…→교회→재림)과 복음을 만나고 — 영접 기도까지 이르게 하는, 무료·공유 가능한 단일 페이지.**

종이 전도지가 4페이지에 복음을 요약한다면, 이 사이트는 **성경 전체의 줄거리를 한 스크롤에 담고, 모든 인용을 진짜 성경 원문에 링크로 연결한 “살아있는 전도지”** 입니다.

- 성경 인용은 **각 언어 공인 번역본 그대로(verbatim)** — 의역하지 않으며, 모든 구절은 탭하면 그 언어의 성경으로 이동합니다.
- 대상은 둘: **“성경을 알지만 통독은 못 해본 사람” + “처음 듣는 사람.”**
- 브라우저(OS) 언어를 자동 감지하고 우측 상단 🌐에서 전환할 수 있으며, **160여 개 언어**로 제공됩니다(정확한 현재 목록은 메뉴).

## 무엇을 담았나
- **13개 장면** — 창조 → 타락 → 족장 → 출애굽 → 정복·사사 → 통일왕국 → 분열왕국 → 포로 → 귀환 → 침묵기 → 예수님 → 교회 → 회복(재림).
- 각 장면: 핵심 인물·사건, 핵심 구절, **‘그리스도 실’**(그 장면이 예수님을 어떻게 가리키는가), **‘멈추지 않는 사랑’**, 그리고 ‘흔한 오해 → 사실은’.
- **복음의 핵심** — 원죄·율법·은혜·이신칭의·하나님 나라·복음; ‘거울’(이스라엘 = 나); **“바로 나 때문에”(십자가)**; 거부감 FAQ(공정성/값싼 용서/“착하게 살면 되지 않나”); **영접 기도**; 그리고 **다음 걸음**(계속 기도 · 성경 읽기 · 가까운 교회).
- 그리스도의 삼중직(참 선지자·제사장·왕).

## 성경 구절을 어떻게 대응하나
우리가 절대 깨지 않는 한 가지 원칙: **성경 구절을 지어내거나 우리가 직접 번역하지 않습니다.** 인용되는 모든 구절은 **공인 번역본 그대로(verbatim)** 이며, 실제 성경(주로 [YouVersion](https://www.bible.com))에서 가져와 글자 단위로 대조합니다. 언어마다 대표 번역본을 씁니다(개역개정·ESV·和合本·新共同訳·RVR1960·Almeida·Louis Segond·Lutherbibel·Синодальный·Van Dyck 등). 줄거리·해설·기도문 같은 **산문은** 언어별로 번역·원어민 검수하지만, **따옴표로 인용되는 본문은 공인 번역본 원문 그대로**입니다.

완역이 없는 언어는 — 성경을 지어내는 일 없이 — 단계적으로 대응합니다:

1. **있는 것은 그대로 인용.** 많은 언어가 신약(때로 구약 일부)을 가지고 있고, 그 본문은 그대로 인용합니다. *예: 키르기스어는 신약 + 창세기 + 사사기가 있어 창조·타락·약속·사사 장면은 실제 키르기스어 성경을 씁니다.*
2. **없는 구약 구절은 신약으로 대체.** 구약 핵심 구절이 없으면 같은 진리의 신약 구절로 바꾸고(예: 이사야 53:5 → 베드로전서 2:24) — 역시 verbatim — 그 외에는 출처를 비운 채 **따옴표 없는 요약**으로 풉니다(깨진 링크 없음).
3. **신뢰 출처 확대.** “유버전에 없다 ≠ 텍스트가 없다” — eBible.org·Digital Bible Library(api.bible)·unfoldingWord/Door43도 활용합니다(성경이 전혀 없는 언어엔 **Open Bible Stories** 포함).
4. **브릿지 언어.** 그 언어 자체에 성경이 없으면 **줄거리·복음은 모국어로** 번역하고, **인용 구절은 화자들이 읽는 지역·국가 공용어로** 보여주며 링크·표기합니다.
5. **그래도 안 되면 정직하게 보류.** 위 어느 것도 아직 불가능하면 *준비 중*으로 표시하고, 번역·출처·도구가 나아지는 대로 다시 봅니다.

현재 다섯 단계가 모두 가동 중입니다 — 완역; 부분·리처부분(키르기스·풀라); eBible 소스 완역(티베트어); Open Bible Stories(발로치어); 브릿지(보즈푸리←힌디·치타고니아←벵골).

**규모보다 정확성.** 목표는 모든 민족의 모국어이지만, 한 언어는 그 번역된 *산문*(줄거리·FAQ·기도문 — AI가 생성하는 부분; 구절은 verbatim 복사)이 **검증 가능하게 충실할 때만** 공개됩니다: 역번역이 원문과 일치 **+** 원어민 검수에 미해결 오류 없음. 충실함이 확인되지 않으면, 화자가 아무리 많아도 **보류**합니다. *지어낸 구절 — 또는 부실한 AI 번역 — 보다 정직한 “아직”이 낫습니다.* 그래서 현실적 도달 범위는 ~7,400개 전부가 아니라 “쓸만한 기계번역이 가능한 ~200개” 부근이며, 이는 의도된 것입니다.(인구 도달률은 큰 언어들로 이미 ~90%.)

## 관점
한국 개신교 대다수가 공유하는 **복음주의·개혁주의 구속사(救贖史)** 관점에 기반합니다.

## 기술 / 배포
- 의존성 없는 **단일 `index.html`**(HTML·CSS·바닐라 JS) + 언어 팩 `i18n/<code>.json`(필요 시 로드).
- **언어별 정적 페이지**(`/en/`, `/ja/`, `/ar/` …): 각 페이지에 그 언어의 `title`·OG(제목·설명·이미지·locale)·canonical·**hreflang** + **현지어 본문 프리렌더** + JSON-LD + llms.txt가 박혀 있어, 공유 시 해당 언어 미리보기(이미지 포함)가 뜨고 검색·AI 크롤러가 언어별로 색인합니다.
- **생성기** `node tools/build-pages.mjs` 가 이 페이지들과 `sitemap.xml`·`llms.txt`를 만듭니다. **Vercel이 매 배포마다 실행**하므로 생성 페이지는 git에 커밋하지 않습니다(커밋되는 바이너리는 공용 OG 이미지·PWA 아이콘·언어별 QR).
- 새 언어 추가는 `/add-language` 스킬을 따릅니다(모드 판정·검증·구절링크 감사·verbatim/산문 검수).
- **Vercel** 자동 배포(`main` 푸시 시 갱신). `robots.txt`·sitemap·OG/JSON-LD·오프라인 PWA 포함.

## 함께 번역하기 (Contribute) — 다 같이, 모든 언어로
이 일은 **다 같이** 할 때 빨라집니다. 당신의 언어를 더하거나 다듬어 주세요. **코딩 지식은 필요 없고, 번역(텍스트)만** 손보면 됩니다 — 빌드·이미지·구절 링크·배포는 관리자가 합니다.

- ✏️ **문구 고치기 (제일 쉬움)**: GitHub에서 `i18n/<언어코드>.json` 을 열고 연필(Edit) → 문장 수정 → *Propose changes* → *Pull request*. 끝.
- 🌍 **새 언어 추가**: `i18n/en.json` 을 복사해 `i18n/<코드>.json` 으로 만들고 값을 번역 → Pull request.
- 📜 **딱 두 가지만**: ① 성경 인용은 **당신 언어 공인 번역본 그대로(verbatim)** — 의역 금지(실제 성경에서 복사). ② **키·HTML 태그·구조는 그대로**, 값(문장)만 번역.

자세한 단계는 **[CONTRIBUTING.ko.md](CONTRIBUTING.ko.md)** (한국어) · **[CONTRIBUTING.md](CONTRIBUTING.md)** (English). 질문은 [이슈](../../issues)로.

## 비슷한 취지의 다른 프로젝트들
많은 사역이 같은 마음을 품고 있습니다. **우리는 그 앞단의 ‘입구’** 이고, 그들에게 링크로 연결합니다. 초점별로 몇 가지:

**큰 그림 개관** (우리와 가장 가까움)
- [BibleProject](https://bibleproject.com) — 성경 전체 줄거리·주제를 애니메이션으로. 다국어 자막, 무료. *(영상 중심, 영접 기도는 없음)*
- [Open Bible Stories — unfoldingWord](https://www.unfoldingword.org/obs) — 창조→회복 성경 이야기 50편, 수백 개 언어 CC 라이선스.

**복음 제시 + 결단** (우리 후반부와 가까움)
- [Peace With God](https://peacewithgod.net) (BGEA) — 복음 + 영접 기도, 다국어.
- [GodTools](https://godtoolsapp.com) (Cru) — 사영리 / 하나님을 인격적으로 아는 것, 약 119개 언어.
- [KnowGod](https://knowgod.com) · [EveryStudent](https://www.everystudent.com) · [StartingWithGod](https://www.startingwithgod.com) (Cru) — 전도 사이트.

**영상·구술 전도** (모든 언어 지향)
- [Jesus Film Project](https://www.jesusfilm.org) — 예수 영화, 2,000+ 언어.
- [God's Story Project](https://www.gods-story.org) · [Create International](https://www.createinternational.com) · Wordless Book — 구술·그림 복음.

**전체 성경 접근** (우리가 링크하는 인프라)
- [YouVersion / Bible.com](https://www.bible.com) — 2,000+ 언어 성경 읽기.
- [Wycliffe](https://www.wycliffe.org) · [SIL](https://www.sil.org) · [ETEN / illumiNations](https://illuminations.bible) — 완역 번역·배포.
